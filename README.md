# ViveController-RTM-pkg

## 1. 概要
VRデバイスの1つであるHTC社のVIVEを対象として，その機能の利用を可能にするRTコンポーネント

## 2. 開発環境
### 2.1. 開発環境一覧
PC : ZBOX-EN1070-U
```
OS : Windows 10
ソフトウェア：CMake / OpenRTM-aist-1.2.0-RELEASE
ハードウェア：HTC VIVE / HTC VIVE Tracker 2018
開発言語：C/C++


## 3. 開発コンポーネントの概要
### ViveController
本コンポーネントは以下の4つの機能をもつ
1)VIVEコントローラの位置姿勢/ボタン情報の管理
2)VIVE HMDの位置姿勢の管理
3)VIVEトラッカーの位置姿勢の管理
4)VIVE HMDへの画像の出力
